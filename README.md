# 아이랑 세종 AI

0-36개월 영유아 가족을 위해 세종시 공공자원, 돌봄 시설, 공식 안내 정보를 조건별로 추천하는 심사용 웹앱입니다.

## 공개 데모

- GitHub Pages: https://gunheeyyes-ui.github.io/sejong-airang-care-ai/

## 핵심 화면

- 오늘 추천: 월령, 생활권, 날씨, 미세먼지, 유모차/수유실/기저귀대 조건을 반영한 공공자원 추천
- 공식정보 Q&A: 임신·출산·육아 관련 질문을 공식 출처 기반 답변으로 정리
- 행정 대시보드: 비식별 이용 흐름, 인기 조건, 추천 수요를 행정 관점으로 요약
